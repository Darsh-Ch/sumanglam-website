# Target Users

## Homeowners

Goals:

* Gather inspiration.
* Explore kitchen ideas.
* Understand available solutions.
* Discover premium brands.
* Visit the showroom.
* Schedule a consultation.

Implementation implications:

* Prioritize visual browsing.
* Keep CTAs clear.
* Avoid technical overload before inspiration.
* Make mobile navigation and WhatsApp access easy.

## Renovation Customers

Goals:

* Upgrade kitchens.
* Upgrade wardrobes.
* Explore appliances.
* Compare premium options.
* Obtain guidance.

Implementation implications:

* Make solution categories clear.
* Link inspiration to brands/products and consultation.

## Architects, Interior Designers, And Builders

Goals:

* Explore brand offerings.
* Evaluate technical solutions.
* Collaborate with Sumanglam.
* Request consultations or technical resources.

Implementation implications:

* Provide professional trust signals.
* Support technical catalog requests or discussion CTAs.
* Do not build Architect Portal in V1.

## Product Researchers

Goals:

* Explore hardware.
* Explore appliances.
* Compare specifications.
* Understand available options.

Implementation implications:

* Product listing and PDPs should support basic category/brand/spec exploration.
* Advanced filtering is not V1 unless explicitly approved.
