# Wardrobes

## Purpose

Present wardrobe and storage systems as part of the Mrida ecosystem, not as an independent product catalog.

## User Goal

Explore wardrobe inspiration and understand Mrida's customized wardrobe/storage solutions.

## Entry Points

* [[Mrida]]
* Main navigation if approved.
* Inspiration collections.
* Homepage Explore Spaces.

## Exit Points

* Mrida wardrobe inspirations.
* [[Mrida]]
* [[Book Consultation]]
* [[Showroom Experience]]

## UI Requirements

* Wardrobe inspiration and storage solution visuals.
* Mrida brand framing.
* Consultation CTA.
* Related collections such as Walk-In Wardrobes, Minimal Wardrobes, Luxury Wardrobes, Storage Solutions.

## UX Behavior

* Always frame wardrobe content through Mrida.
* Avoid SKU/product-first browsing for wardrobes.
* Use inspiration-first discovery.

## Content Requirements

* Mrida wardrobe story.
* Wardrobe inspiration imagery.
* Storage solution content.
* Related consultation copy.

Public page.

## Edge Cases

* If route exists, nav label must not imply standalone product catalog.
* Missing Mrida wardrobe assets.

## Forbidden Mistakes

* Do not model wardrobes as independent product catalogs.
* Do not imply Nolte wardrobes.
* Do not create wardrobe product taxonomy without explicit approval.
