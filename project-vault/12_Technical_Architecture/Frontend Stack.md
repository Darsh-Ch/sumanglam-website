# Frontend Stack

## Framework

Next.js 15.

## Routing

App Router.

## Language

TypeScript.

## Styling

Tailwind CSS.

## Component System

shadcn/ui.

## Animation

Documented:

* Framer Motion.

User-approved extension:

* Three.js.
* GSAP with ScrollTrigger.
* Lenis.js.

## State Management

* Prefer React Server Components.
* Use minimal client-side state.
* Use Zustand only if necessary.
* Do not use Redux.

## Implementation Rules

* Mobile-first.
* Use Server Components by default.
* Use Client Components only where interactivity requires them.
* Keep component boundaries small and reusable.
