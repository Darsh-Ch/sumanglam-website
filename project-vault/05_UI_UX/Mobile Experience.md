# Mobile Experience

## Priority

Mobile is the primary experience. Desktop extends mobile; desktop does not define mobile.

## Requirements

* Fast loading.
* Large imagery.
* Minimal text.
* Simple navigation.
* Accessible CTAs.
* Smooth scrolling experience.
* Thumb-friendly navigation.
* Persistent WhatsApp access.
* Clear CTA visibility.
* No zoom required.

## Implementation Guidance

* Design each section for mobile first.
* Avoid dense grids on mobile.
* Keep CTAs reachable without hunting.
* Use responsive images and Cloudinary transformations.
* Respect reduced-motion preferences.
* Do not hide critical actions behind hover-only interactions.

## Forbidden Mistakes

* Do not design desktop-first.
* Do not hide critical actions on mobile.
* Do not ship unoptimized images.
* Do not overfill mobile sections with text.
