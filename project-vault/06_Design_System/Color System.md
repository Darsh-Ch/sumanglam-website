# Color System

## Current Status

The source docs define emotional direction and anti-patterns, but do not specify exact color tokens.

## Required Mood

Colors should support:

* Luxury.
* Trust.
* Warmth.
* Modernity.
* Sophistication.
* Approachability.

## Forbidden Color Directions

* Neon colors.
* Cyberpunk palettes.
* Dark tech startup themes.
* Excessive gradients.
* Over-saturated colors.

## Implementation Guidance

Define tokens before building final UI:

* Background.
* Surface.
* Text primary.
* Text secondary.
* Border.
* Accent.
* CTA primary.
* CTA secondary.
* Error/success/warning.

## Open Question

Exact brand colors are not specified. This blocks final design system decisions.
