# Spacing Layout

## Whitespace Philosophy

Whitespace is a feature, not empty space.

## Layout Rules

* Use generous spacing.
* Allow sections to breathe.
* Avoid visual crowding.
* Avoid dense layouts.
* Use large blocks.
* Use strong hierarchy.
* Use clear focal points.
* Use intentional composition.

## Avoid

* Tiny cards everywhere.
* Overloaded grids.
* Visual clutter.
* Dense catalog layouts.

## Implementation Guidance

* Create reusable Container and Section components.
* Keep section rhythm consistent.
* Use responsive spacing tokens.
* Adapt desktop layouts from mobile-first structure.
* Do not nest decorative cards inside cards.
