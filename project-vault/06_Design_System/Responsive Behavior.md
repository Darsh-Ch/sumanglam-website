# Responsive Behavior

## Principle

Mobile first. Desktop extends mobile.

## Mobile Priorities

* Large imagery.
* Minimal text.
* Fast loading.
* Thumb-friendly navigation.
* Accessible CTAs.
* Persistent WhatsApp access.

## Desktop Priorities

* Generous whitespace.
* Large imagery.
* Editorial layouts.
* Sophisticated scrolling experience.
* No overcrowding.

## Implementation Guidance

* Start with mobile layout and progressively enhance.
* Avoid hover-only critical interactions.
* Keep CTAs visible and touch-friendly.
* Use responsive images.
* Test content wrapping in cards, buttons, nav, and forms.
