# Image Strategy

## Purpose

Images are the primary storytelling tool for the platform.

## Homepage Priority

1. Professional kitchen visuals.
2. Official brand assets.
3. High-quality renders.
4. Professionally edited showroom photography.

## Global Image Priority

1. Professional interiors.
2. Kitchen environments.
3. Wardrobes.
4. Product close-ups.
5. Showroom imagery.

## Technical Requirements

* Use Next.js Image.
* Use Cloudinary transformations.
* Use responsive image delivery.
* Avoid full-resolution original image delivery.

## Avoid

* Flash photography.
* Over-saturated colors.
* Busy compositions.
* Low-quality phone photography in hero sections.
* Stock-photo aesthetics.
* Unoptimized image tags.
