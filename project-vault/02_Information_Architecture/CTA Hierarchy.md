# CTA Hierarchy

## Priority Order

1. Explore Inspirations
2. Visit Showroom
3. Book Consultation
4. WhatsApp Inquiry
5. Contact Us

## Homepage CTA Use

Primary CTA:

* Explore Inspirations

Secondary CTA:

* Visit Showroom

Tertiary CTA:

* Book Consultation

Supporting CTA:

* WhatsApp Inquiry

## Conversion Principles

* CTAs should guide users from inspiration to engagement.
* CTAs should not feel aggressive.
* Consultation CTAs should be problem-focused, not sales-focused.
* Mobile users should have persistent access to WhatsApp where appropriate.
